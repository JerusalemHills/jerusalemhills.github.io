‚ùì Frequently Asked Questions

Common Questions

Q: Can multiple children use the same device?

A: Yes! Each browser profile maintains separate progress data. Consider creating separate browser profiles for each child.

Q: What happens if we clear browser data?

A: Progress will reset, but children can re-earn badges. The learning experience remains valuable regardless of tracking.

Q: Can teachers access student progress?

A: No. Only the person using the device can view progress. This ensures privacy while still providing motivation.

Q: Are there any reports for parents?

A: The progress dashboard shows comprehensive achievement data. We're working on printable summary reports.
