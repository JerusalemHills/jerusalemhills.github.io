  • Personal Information: No names, emails, addresses, or contact details stored
  • User Tracking: No logs, tracking cookies, or behavioral monitoring
  • Game Progress: No permanent storage of game scores, progress, or preferences
  • Forum Posts: No server-side storage of user posts or messages
  • Browsing History: No tracking of pages visited or time spent
  • Device Information: No fingerprinting or device identification

Ephemeral & Temporary Data Only:

Any information that may be displayed on the site that is user-related (such as game state, session progress, or temporary interactions) is entirely ephemeral and temporary.

  • Browser Session Only: Data exists only during your current browser session
  • No Server Storage: Game states, scores, and preferences stored locally in browser memory only
  • Automatic Deletion: All temporary data is automatically cleared when you close your browser
  • No Logs: We maintain no server logs of user activities or interactions
  • No Backups: No user-related data is ever backed up or archived

Technical Architecture for Privacy:

  • Forum: Uses GunDB decentralized P2P network - no central server data storage
  • Games: All game logic runs in your browser - no server communication required
  • Static Hosting: GitHub Pages static hosting with no server-side processing
  • Minimal Analytics: Only basic Google Analytics for aggregate site improvement (no individual tracking)

What This Means for You:

  • Your privacy is protected by design, not just by policy
  • No data breaches are possible because we don't store your data
  • No targeted advertising or tracking networks
  • No user profiles or behavior analysis
  • Complete anonymity in all interactions
Privacy by Design: Our commitment to zero data collection is implemented through our technical architecture. We cannot track you because our systems are designed not to collect or store user information in the first place.
